How to Setup This Test Page

  1. Drop your OVV-integrated VPAID unit into this directory and name it ad.swf. Alternatively, alter xml/vast.xml to point to your OVV-integrated VPAID unit.
  2. Within your VPAID unit, use ExternalInterface to call 
    displayEvent()
    , for example: 
    	ExternalInterface.call("displayEvent", vpaidEvent.type, ovvCheck.percentViewable, ovvCheck.focus, ovvCheck.viewabilityState);
  3. (Optional) Drop a content video into this directory and name it content.flv. Alternatively, alter the flashvars.file attribute in js/viewability.js to point to your content video.
